Typisierung primitiver Datentypen

// JavaScript: Duck-Typing
var myCount = 4;

function square(count) {
  return count * count;
}

alert("Result: " + square(myCount)); // "Result: 16"
// TypeScript
var myCount: number = 4;

function square(count:number):number {
  return count * count;
}

alert(`Result: ${square(myCount)}`); // "Result: 16"

Update von Angular 1 auf 2

  • Scope und Controller entfernt

    Modularer Aufbau aus Komponenten

  • Two-Way-Databinding verfeinert

    Bestimmung der Richtung des Bindings

  • Verwendung beider Frameworks zugleich

    Inkrementelles Upgrade möglich

Fazit

  • Einsatz von TypeScript verbessert Code-Qualität

  • Angular 2 hat großes Potential als JavaScript Framework

  • Upgrade von Angular 1.x auf 2.0 lohnenswert, da inkrementell möglich

  • Komponenten-Hierarchie stellt nachhaltige und wartbare Struktur dar

  • Angular 2 + TypeScript zu empfehlen (-> gute Dokumentation)
